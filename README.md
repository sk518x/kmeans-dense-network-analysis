# K-Means Dense Network Analysis

Overview

This project applies K-Means clustering to a dense network of 600 nodes. Each node has X and Y coordinates and a random energy value.


Observation

K-Means groups nodes effectively, but results depend on the chosen k value and do not consider energy.


Files and Folders Description

-kmeans.py - the main Python script.
-cmd-run.png - screenshot of the CMD execution.
-output-k7 - contains the output plots for K = 7 clusters.  
-output-k3 - contains the output plots for K = 3 clusters.
