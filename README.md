# K-Means Dense Network Analysis

### Overview

This project applies K-Means clustering to a dense network of 600 nodes. Each node has X and Y coordinates and a random energy value.

### Observations

- K-Means clusters nodes effectively based on spatial positions.
- The cluster assignments depend on the chosen k value.
- Energy values are not used in the clustering, but are available for analysis in the plots.
- Critical nodes can be identified from the energy-based visualizations.

### Files and Folders Description

- kmeans.py - Main Python script implementing K-Means clustering and generating plots.
- cmd-run.png - Screenshot showing CMD execution of the script.
- output-k7 - Contains output plots generated with K = 7 clusters.
- output-k3 - Contains output plots generated with K = 3 clusters.
